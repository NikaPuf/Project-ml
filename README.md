# Project-ml
*Основная идея:* Для начала собираем данные с разных топовых иностранных вузов, после чего платформа предсказывает реальные шансы абитуриента поступить в тот или иной университет с небольшими советами по подготовке

*Machine Learning:* Модель, которая будет оценивать вероятность поступления абитуриента. Входными данными будут данные студента (языковые тесты, дополнительные экзамены, волонтерство (доп достижение), gpa студента); данные каждого вуза (рейтинг, страна, требования, статистика приема). Буду использовать логистическую регрессию.
