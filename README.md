# Project-ml
*Основная идея:* Для начала собираем данные с разных топовых иностранных вузов, после чего платформа предсказывает реальные шансы абитуриента поступить на ту или иную программу по его достижениям и помогает выстроить стратегию подготовки к поступлению на основе профиля.

*Machine Learning:* Модель, которая будет оценивать вероятность поступления абитуриента. Входными данными будут данные студента (языковые тесты, опыт, желаемая страна поступления, gpa студента); данные каждого вуза (рейтинг, страна, требования, статистика приема). Буду использовать логистическую регрессию
